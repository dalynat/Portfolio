Bio

Hello, I am Nathan Daly and this is my portfolio page! I am going to start off by telling you a little background on me. I started off working in restaurants as a chef, but after a couple years, I realized that I did not want to do that anymore so I started looking for a change! When I started looking for a career alternative, I found that coding could be a great fit for me, so I started learning on my own. I played around on Code Academy and bought a course on Udemy to teach myself. At this point I knew that I could see myself working as a software developer, but knew that I had no where near enough knowledge to land a job. I started looking into schooling and found that Penn LPS hosted a 24 week coding bootcamp for students to jumpstart their careers. Here I am 24 weeks later, and I have learned so much. The course taught me Javascript, Bootstrap, React, MySQL, MongoDB, and so much more. Below, I have provided a list of some of my projects and descriptions for each, as well as links to the deployed project and to the code that makes them work.

Projects

Mickeys Pizza:

For this project I worked in a group to develop a pizza store website using React.js, MongoDB, Node.js, Passport.js, Bootstrap, and our own CSS. My role in the project was to use React to make all of the forms for incoming user information, make post and get requests to store and retrieve data from the database, and create user authentication using Passport.
